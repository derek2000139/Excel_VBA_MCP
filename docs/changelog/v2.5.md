# ExcelForge v2.5 变更记录

**发布日期**：2026-03-30
**版本**：v2.5.0
**前置版本**：v2.4.0

---

## 重大更新

### 1. Host 多执行模式正式落地

ExcelForge 现在支持三种执行模式：

| 模式 | 说明 | 需要的组件 |
|------|------|-----------|
| `runtime` | Excel COM 自动化 | Excel + Runtime |
| `file_package` | 直接文件包操作 | 无（纯 Python） |
| `batch` | 批量文件处理 | 无（纯 Python） |

**影响**：所有现有 `runtime` 工具继续正常工作；新增工具默认使用 `file_package` 模式，无需启动 Excel。

## 配置文件变更

### bundles.yaml

- 填充 `artifact_export` bundle（原来 `tools: []`）
- 新增 `artifact_patch`、`batch_ops`、`format_rules` bundle
- 更新 `chart` domain（原来 `tools: []`，现在 4 个工具）
- 所有新 domain 设置 `execution_mode: file_package`

### profiles.yaml

- 新增 `artifact_extract`、`artifact_transform`、`batch_delivery` Profile
- 更新 `calc_format`（+4 工具 → 30）
- 更新 `reporting`（+4 工具 → 24）

### profiles.dev.yaml

- 新增 5 个开发 Profile

### host.py

- 新增工具中文 schema（26 个工具）
- 扩充 TOOL_MANIFEST_MAP
- 支持 mixed-mode 执行链路分发
